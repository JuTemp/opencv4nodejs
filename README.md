# opencv4nodejs

Upstream: [@u4/opencv4nodejs](https://www.npmjs.com/package/@u4/opencv4nodejs)

# Note

Do not override the internal `node_modules` with external `node_modules`, even reinstalling `@u4/opencv4nodejs` doesn't seem to work.

You can copy the internal `node_modules` out first and then install the ones you need, because its dependencies seem to be very complex and fragile.
